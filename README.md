# Quick Chat App 💬

Quick Chat App is a full-stack, real-time messaging application designed for seamless and instant communication. Built with performance and scalability in mind, it provides a responsive interface and dynamic real-time data updates.

## ✨ Features

* **Real-time Messaging:** Instant message delivery using WebSockets (Socket.io).
* **User Authentication:** Secure signup and login functionality (JWT based).
* **Responsive UI:** A clean, modern, and fully responsive chat interface built with React.
* **Active Status:** See when other users are online.
* **Persistent Storage:** Chat history and user data securely stored in MongoDB.
* **Fast Development Server:** Frontend powered by Vite for rapid hot-module replacement and optimized builds.

## 🛠️ Tech Stack

**Frontend:**
* React.js
* Vite
* Tailwind CSS 
* Socket.io-client

**Backend:**
* Node.js
* Express.js
* MongoDB & Mongoose
* Socket.io
* JSON Web Tokens (JWT) for Authentication
